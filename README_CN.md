![Tango Logo](logo.png)

Tango 是一个微内核易扩展的Go语言Web框架.

##### 当前版本: v0.4.6   [版本更新记录](https://github.com/lunny/tango/releases)

## 特性

- 强大而灵活的路由设计
- 兼容已有的`http.Handler`
- 模块化设计，可以很容易写出自己的中间件
- 高性能的依赖注入方式
- 丰富的中间件实现AOP编程风格，灵活定制框架功能

## 安装Tango：
    go get github.com/lunny/tango
